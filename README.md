# Projet-Integrateur-NexaLab-
Projet intégrateur de développement web (IFRI - Bloc 2). Création complète et de zéro de la page d'accueil de NexaLab, un laboratoire d'innovation technologique. Conçu de manière 100% sémantique avec une architecture CSS modulaire, une mise en page Flexbox, le respect de la charte graphique "Cyber Nuit" et strictement sans JavaScript.
